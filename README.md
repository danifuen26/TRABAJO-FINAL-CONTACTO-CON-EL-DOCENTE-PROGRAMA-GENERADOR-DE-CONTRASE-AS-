# TRABAJO-FINAL-CONTACTO-CON-EL-DOCENTE-PROGRAMA-GENERADOR-DE-CONTRASE-AS-

Programa generador de contraseñas seguras

ESTUDIANTE : FUENTES PAREDES DANIELA CCAROLINA

OBJETIVO DEL SISTEMA :

Desarrollar un generador seguro de contraseñas que permita a los usuarios crear contraseñas fuertes y confiables de manera rápida y sencilla, proporcionando una alternativa que contribuya a mejorar la seguridad de sus cuentas y la protección de su información personal.
DESCRIPCION DE FUNCIONALIDADES: 

Descripción de funcionalidades

Al iniciar el programa, se muestra una breve presentación del sistema y del desarrollador, proporcionando al usuario una bienvenida e introducción al funcionamiento de la aplicación.

Importación de librerías: Se utilizan las librerías `random` y `string` para generar contraseñas aleatorias utilizando letras mayúsculas, minúsculas, números y símbolos especiales. Además, se define el archivo donde se almacenarán las contraseñas generadas.

Generación de contraseñas: Esta función crea contraseñas seguras combinando diferentes tipos de caracteres de forma aleatoria. El usuario puede especificar la longitud deseada y el sistema genera una contraseña con un alto nivel de seguridad.

Almacenamiento de contraseñas (guardar contrasena):Permite guardar las contraseñas generadas en un archivo de texto junto con el nombre del sitio o servicio correspondiente. La información se agrega al archivo sin eliminar los registros existentes.

Consulta de contraseñas (buscar contrasenas) Muestra todas las contraseñas previamente almacenadas. En caso de que el archivo aún no exista, el programa informa al usuario mediante un mensaje amigable, evitando errores durante la ejecución.

Menú principal (main): Implementa un menú interactivo que permanece en ejecución hasta que el usuario decide salir. Desde este menú es posible generar una contraseña, visualizar las contraseñas guardadas o finalizar el programa. Además, valida las opciones ingresadas para evitar entradas incorrectas.

Opción de generación:Solicita al usuario la longitud de la contraseña y emite una advertencia si esta es menor a ocho caracteres, ya que una longitud reducida disminuye el nivel de seguridad. Posteriormente, genera la contraseña y ofrece la posibilidad de guardarla.

Opción de consulta y salida: La segunda opción permite visualizar todas las contraseñas almacenadas, mientras que la tercera finaliza la ejecución del programa mostrando un mensaje de despedida.

En conjunto, el sistema integra funciones de generación, almacenamiento y consulta 
de contraseñas, incorporando validación de entradas, manejo básico de errores y persistencia de datos mediante archivos, lo que lo convierte en una solución práctica, sencilla y funcional para la gestión de contraseñas seguras.
